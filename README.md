# SOCL-Web-app

If during migrations, there are issues, do this:  
> mysql -u root -p  
> delete database socl;  
> create database socl;  
  
Install pip (8.1.2) and venv and then activate venv (source <venv>/bin/activate)  
Then run the following:  
> pip install requirements.txt  

Format:  
app_name/stuff/edit  
app_name/stuff/show  
app_name/stuff/delete  
app_name/stuff/add
